Exercise 7.4: anagrams

Given a dictionary, write all anagrams of an input string.

Naive strategy: write all permutations of the string, split it in any possible way, then check all the word are in the dictionary.

We can improve efficiency using a data structure called Trie. It is a dictionary of string that has the operation startWith(String prefix) that return true iff exists a word in the dictionary starting with the given prefix.
