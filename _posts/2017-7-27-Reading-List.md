The list of book I'm reading, I've read or I'm going to read as soon as possible. As I lose interest in a particular topic, I remove the books associated with it.

## Type theory

* [Lectures on the Curry-Howard Isomorphism, Sørensen, Urzyczyn](https://www.elsevier.com/books/lectures-on-the-curry-howard-isomorphism/sorensen/978-0-444-52077-7)
    - Intuitionistic logic, BHK interpretation
    - Heyting algebra
    - Dependent types, system λP
    - Pure type systems
* (Course) [Type Theory (Spring 2005, Herman Geuvers)](http://www.cs.ru.nl/~herman/TT/)
    - Polymorphic types
    - Dependent types
    - λHOL
    - Coq (practice)
* [Homotopy type theory: Univalent Foundations of Mathematics](https://homotopytypetheory.org/book/)
    - Type theory vs set theory
    - still todo

## Category theory
* [Category theory, Awodey](https://www.amazon.com/Category-Theory-Oxford-Logic-Guides/dp/0199237182)
    - Very clever introduction to category theory
* Topoi - The categorical analysis of logic, Goldblatt
    - Book that focuses on topoi, contains an overview of category theory that seems easier than reading the whole Awodey's one
* (Blog + Youtube videos) [Abstract Nonsense](https://abstract-nonsense.com/category-theory/)
    - Functors, Natural transformation, Adjoint
    - Other stuff related (algebra, topology, ...)
